## 💡 简介

基于思源笔记修改而来的个人笔记软件，一款本地优先的个人知识管理系统， 支持细粒度块级引用和 Markdown 所见即所得。

## 🚀 发布

首先在分支上创建一个标签，然后推送标签触发CI和docker镜像构建，标签名称示例：v2.0.27

```bash
git tag v2.0.27
git push origin v2.0.27
```
